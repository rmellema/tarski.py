tarski.world module¶

This module contains all the code for representing the blocks world in Python.

class tarski.world.Block(identifier, size, shape, x, y)[source]¶

Bases: object

Describes a block in the Block World. It has the following attributes, which all have to be set when the block is initialized.

identifier¶

An unique identifier for this block. The creator of the Block must make sure it is unique.

shape¶

The shape of this Block.

size¶

The size of this block

x¶

The x location of this block on the board

y¶

The y location of this block on the board

class tarski.world.World(x_size=8, y_size=8)[source]¶

Bases: object

The world that holds all of the actual blocks. It has the following attributes:

x_size¶

The size of the board in the x direction. Has to be set at initialization. Defaults to 8.

y_size¶

The size of the board in the y direction. Has to be set at initialization. Defaults to 8.

domain¶

A set of blocks that symbolize all the blocks that are in this world. It should not be directly altered.
