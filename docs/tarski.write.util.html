tarski.write.util module¶

A collection of utility functions for writing the models.

tarski.write.util.make_pair(coll, lbracket='<', rbracket='>')[source]¶

A context aware function for making a string representation of elements of relationships. It takes into account the length of the element. If there is just one element, the brackets are left of, but when there are more, all the elements will be seperated by a comma, and the brackets will be inserted as well.

Parameters:
  • coll – The collection that needs to be printed. Can be a generator, but cannot be infinite.
  • lbracket (str) – The bracket that goes on the left.
  • rbracket (str) – The bracket that goes on the right.
Returns:

A context aware string representation of the pair.
Return type:

str
