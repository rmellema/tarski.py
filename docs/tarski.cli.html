tarski.cli.convert(files, iformat, output, oformat, use_header=None)[source]¶

Convert all the files in files into the corresponding first order structures and write them to the output.

Parameters:
  • files (list[string]) – A list of file names that need to be converted.
  • output – The stream to write the output to.
  • use_header (bool) – Whether or not to write a header for each file.
